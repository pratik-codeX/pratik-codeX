## 👨‍💻 About Me

I am a Software Developer deeply passionate about **how things work under the hood**. Rather than just building on top of abstractions, I enjoy diving into file systems, network sockets, memory management, and process execution on Linux. 

At **Marvellous Infosystems**, I am rigorously building strong CS fundamentals and applying them through system-level C/C++ utilities and scalable Java/Python backend systems.

**Currently Focusing On:**
- 🐧 **Linux Internals** (Process Management, Virtual Memory, `/proc` filesystem)
- 🔌 **Network Programming** (TCP/UDP Socket architecture, Concurrent Servers)
- 🏗️ **Backend Architecture** (RESTful APIs, Spring Boot, Microservices)
- ⚙️ **Performance & Debugging** (Writing memory-safe, optimized C/C++ code)
